Kaynaklar

Lütfen öncelikle Kurulumda'ki adımları tamamlayın.

Datayı oluşturma

  1. HuTao-GD Github

  2. npm run build Komutunu kullanın

  3. Kaynakları ./InputData/{version}/ 'un içine koyun

    HuTao-GD

    ├─InputData

    │ └─X.X.X

    │──├─BinOutput

    │──├─ExcelBinOutput

    │──└─Scripts

  4. RunConvert.bat'ı açın

  5. ./OutputData/{version}/* içindekileri buraya yerleştirin -> (HuTaoGS)/data/game/{version}/

Protoları kurma

  1. HuTao-GD-Protos'u Discord/#resources'dan indirin Discord link

  2. Proto Dosyalarını ./data/proto/*'nun içine atın.

  3. HuTao-GS

    ├─data

    │ └─Proto

    │──├─X.X.X

    │──├─ForceUpdateInfo.proto

    │──├─PacketHead.proto

    │──├─QueryCurrRegionHttpRsp.proto

    │──├─QueryRegionListHttpRsp.proto

    │──├─RegionInfo.proto

    │──├─RegionSimpleInfo.proto

    │──├─ResVersionConfig.proto

    │──└─StopServerInfo.proto

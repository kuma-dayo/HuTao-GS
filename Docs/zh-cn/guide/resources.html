Resources

请先完成 构建 的步骤再继续。

生成 data

  1. 下载 HuTao-GD Github

  2. 安装以下依赖项

    npm install

    npm i typescript

    npm i tsc-alias

  3. 运行 npm run build进行构建

  4. 将resources里的文件夹放到 ./InputData/{version}/

    例如：

    HuTao-GD

    ├─InputData

    │ └─X.X.X

    │──├─BinOutput

    │──├─ExcelBinOutput

    │──└─Scripts

  5. 运行 RunConvert.bat

  6. 从 ./OutputData/{version}/* 复制到 (HuTao-GS)/data/game/{version}/

添加 Protos

  1. 从我们的discord #resources 里下载 HuTao-GS的Protos Discord

  2. 将protos放入./data/proto/*

    HuTao-GS

    ├─data

    │ └─Proto

    │──├─X.X.X

    │──├─ForceUpdateInfo.proto

    │──├─PacketHead.proto

    │──├─QueryCurrRegionHttpRsp.proto

    │──├─QueryRegionListHttpRsp.proto

    │──├─RegionInfo.proto

    │──├─RegionSimpleInfo.proto

    │──├─ResVersionConfig.proto

    │──└─StopServerInfo.proto

Resources

Please complete the steps in Building before you continue.

Generate data

  1. Download HuTao-GD on Github

  2. Run npm run build

  3. Put resources in ./InputData/{version}/

    HuTao-GD

    ├─InputData

    │ └─X.X.X

    │──├─BinOutput

    │──├─ExcelBinOutput

    │──└─Scripts

  4. Execute RunConvert.bat

  5. Copy ./OutputData/{version}/* -> (HuTaoGS)/data/game/{version}/

Setup Protos

  1. Download HuTao-GD-Protos in Discord/#resources Discord

  2. Put protos in ./data/proto/*

    HuTao-GS

    ├─data

    │ └─Proto

    │──├─X.X.X

    │──├─ForceUpdateInfo.proto

    │──├─PacketHead.proto

    │──├─QueryCurrRegionHttpRsp.proto

    │──├─QueryRegionListHttpRsp.proto

    │──├─RegionInfo.proto

    │──├─RegionSimpleInfo.proto

    │──├─ResVersionConfig.proto

    │──└─StopServerInfo.proto
